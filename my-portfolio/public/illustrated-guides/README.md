# Illustrated Guides Cover Images

Please add cover images for the illustrated guides:

1. **rlhf-cover.png** - Cover image for RLHF Illustrated Guide
   - Recommended size: 800x600px or 1200x630px
   - Source: Screenshot from https://illustrated-rlhf.vercel.app/

2. **stats-mlops-cover.png** - Cover image for DriftCity: Statistics for MLOps  
   - Recommended size: 800x600px or 1200x630px
   - Source: Screenshot from https://stats-for-mlops.vercel.app/chapters/chapter-1

## How to add images:

1. Take screenshots from the deployed Vercel sites
2. Resize to recommended dimensions
3. Save as PNG format
4. Place in this directory (/public/illustrated-guides/)

The portfolio will automatically display these images on the cards and detail pages.
