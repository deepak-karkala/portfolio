Why this project ?

This project attempts to explain how the seemingly impossible superman position on the bike without pedalling can beat the rest of the competitors pedalling hard. The purpose of the project is to demystify the seemingly impossible phenomena by explaining the basic physics concept in a way that is relatable to most people. The article introduces physics concepts to students in a practical way. Often the way physics taught in schools and universities is unrelatable and seems out of context for most students. In order to make science more interesting, it is better if students can relate better to the concepts taught on paper. This project aims to serve that purpose by using basic concepts of forces acting on a bicycle to explain the unusual aerodynamic position used in bicycle racing
